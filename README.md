# Portfolio Personal

Este es un portafolio personal moderno y responsivo, construido utilizando únicamente tecnologías web estándar: **HTML5**, **CSS3** y **JavaScript**.

## Estructura del Proyecto

- `index.html`: Estructura principal del sitio.
- `styles.css`: Estilos personalizados y diseño responsivo (incluyendo tema oscuro y efecto glassmorphism).
- `script.js`: Lógica para renderizar proyectos, habilidades y manejo de interacciones.

## Características

- **Diseño Responsivo**: Se adapta a móviles, tablets y escritorio.
- **Tema Oscuro**: Diseño elegante con colores oscuros y acentos de color.
- **Glassmorphism**: Efectos modernos de transparencia y desenfoque.
- **Sin Dependencias**: No requiere Node.js, NPM ni pasos de compilación.

## Cómo Ejecutar

Simplemente abre el archivo `index.html` en tu navegador web de preferencia. No se requiere ninguna instalación ni paso de construcción.

## Personalización

- **Contenido**: Puedes editar la información de los proyectos y habilidades directamente en el archivo `script.js`.
- **Estilos**: Los colores y fuentes se pueden ajustar en las variables CSS al inicio de `styles.css`.
